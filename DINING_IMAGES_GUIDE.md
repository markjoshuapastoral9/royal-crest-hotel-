## 🎨 Image Specifications

**Dimensions:**
- Hero images: 1200x500px or 1920x800px

**Format:** JPG or WebP
**File Size:** Max 300KB per image
**Quality:** 80-85%

## 🚀 Quick Setup Instructions

### Method 1: Download from Unsplash
1. Right-click each URL above → "Save image as..."
2. Rename to match required filenames
3. Copy all images to `E:\royal-crest-hotel\public\images\`
4. Refresh browser

### Method 2: Use Placeholder Service (Temporary)
Keep the current code as is - it will show placeholder boxes until you add real images.

### Method 3: AI-Generated (Leonardo.ai)
1. Go to https://leonardo.ai and sign up (free)
2. Use "Phoenix" or "Photoreal" model
3. Generate images with the prompts above
4. Download and rename
5. Copy to `public/images/` folder

## 💡 Pro Tips

1. **Consistency**: Use images with similar color tones (warm, elegant)
2. **Lighting**: Choose well-lit photos that match your hotel's ambiance
3. **Quality**: Avoid pixelated or low-res images
4. **Optimization**: Compress images using TinyPNG.com before uploading
5. **Real Photos**: Best option is to hire a photographer for actual hotel photos (₱5,000-15,000)

## 📞 Need Help?

If images aren't displaying:
1. Check file names match exactly (case-sensitive)
2. Verify files are in `public/images/` folder
3. Clear Laravel cache: `php artisan cache:clear`
4. Check file permissions
